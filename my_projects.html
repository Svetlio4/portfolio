My first web project

My first web project is a fan-site of a football club. In this case it was for Arsenal FC. The site contain a home page, team page, fixtures page and social media page. The home page contains songs of the fans of the club and a description of the fan club. It also contains photos of the fans.

The team page contains a table with the team players and coach. It's devided by the players positions. The page contains and a photo of the team.

The fixtures page contains the following matches of the team with dates and time of the start of the matches.

And the last page is the social media page where u can follow the teams social medias. It contains all links to Arsenal's FC pages for the fans.

The site is not well designed, but it was my first time creating such a project so i gave my best back then.
